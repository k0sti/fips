Connect Your Node

1. Build FIPS

git clone https://github.com/jmcorgan/fips.git
cd fips
cargo build --release

2. Create your config (~/.config/fips/fips.yaml)

node:
  identity:
    nsec: "nsec1..."  # your key, or omit for ephemeral

transports:
  udp:
    bind_addr: "0.0.0.0:4000"

peers:
  - npub: "npub1e2z5mnxk237an7uwca4sxn6mv4a9v66djw9xwsxykf9f42k5ckvqn8z8jz"
    alias: "atlantislabs"
    addresses:
      - transport: udp
        addr: "node.fips.atlantislabs.space:4000"

3. Run

sudo ./target/release/fips

4. Verify connection

fipsctl show peers
fipsctl show links

What is FIPS?

A decentralized mesh routing protocol using Nostr keypairs as node identities. Self-organizing, multi-transport, end-to-end encrypted. No central authority.
